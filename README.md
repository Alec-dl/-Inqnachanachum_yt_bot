## Переменные окружения

| Переменная | Значение |
|---|---|
| TELEGRAM_TOKEN | Токен от @BotFather |
| TELEGRAM_CHANNEL | @inqnachanachum |
| YOUTUBE_API_KEY | AIzaSyAzgG8Ax-KOu28AOy89g8KV9wjDfQ9cgow |
| YOUTUBE_CHANNEL_ID | (получается автоматически) |

## Запуск на Railway.app

1. Зайдите на railway.app
2. New Project → Deploy from GitHub
3. Загрузите эти файлы на GitHub
4. Добавьте переменные окружения в Settings → Variables
5. Бот запустится автоматически!
